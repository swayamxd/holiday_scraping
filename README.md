# holiday_scraping

## scrapes data from [Holidify](https://www.holidify.com/), a travel website

Current extraction features: 

* City Name
* Places to Visit in the city
* State
* Country
* Weather
* Best time
* Ideal duration
* About
* Nearest Airport
* Upcoming events
* More on City
* How to reach

It output's in `csv` format, which can be found here: [holidify.csv](/holidify.csv)
